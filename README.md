## :pencil2:개요
- 바쁜 현대인의 건강을 위한 간편한 온라인 샐러드 플랫폼
- 나만의 샐러드를 직접 제작해 알레르기가 있는 사람들도 안심하고 먹을 수 있는 샐러드 제공
- 메뉴 선택이 어려운 분들을 위한 맛 보장 완제품 샐러드 제공
